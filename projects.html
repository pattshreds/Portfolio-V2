PokéSearch

PokéSearch is built using HTML, CSS, Javascript, and jQuery using a third party API. This front end application allows the user to search a Pokédex-style database to learn information about each Pokémon, and various berries found in the games.

Moodset

Moodset was built using Javascript, NodeJS, Mongoose, ExpressJS, and .EJS in an MVC format. This full stack application allows users to create, display, update, and delete playlists in which they can upload their own mp3 files.

Infinite Savannah

Infinite Savannah was built using a MERN stack (MongoDB, ExpressJS, React, NodeJS). This full stack application allows the user to explore a retro-futuristic skateshop, and create, view, edit, and delete items via a user-friendly interface. Infinite Savannah was pair programmed as a team using Agile methodologies, and Trello to manage progress.

Rezu

Rezu was built using a PERN stack (PostgreSQL, ExpressJS, React, NodeJS). This full stack application is a responsive and easy-to-use product review website that I built as a passion project to practice using complex React techniques and tools, and to learn about implementing a PostgreSQL database in React.
