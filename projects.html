PokéSearch

  • PokéSearch was created using technologies such as HTML, CSS, Javascript and jQuery and a third party Pokémon API.

Moodset

  • This application was developed because of my interest in learning how to create a platform where someone can upload and play music files, and create playlists in a full-stack application.
  • Javascript, Node.JS, Mongoose, Express.JS, .EJS, and Cloudinary were the tools used.

Infinite Savannah

  • Cultivated in a pair-programming environment, Infinite Savannah was made by a colleague and myself. A Kanban board (Trello) was used to organize and track activity.
  • CRUD style application harnessing the capabilities of MongoDB & Mongoose, Express.JS, React, Node.JS.

Rezu

  • With AI and bots being so easy to use today, online reviews can be very misleading. The idea for Rezu hatched to address this common frustration felt by consumers of e-commerce.
  • This initiative employed React, Node.JS, Express.js, and SQL to query a Postgres database. User authentication will be provided by Auth0.
